## Details

- *Typefaces:* `Overpass` (regular & bold)
- *Colours:* `#fff`, `#ff9500`, `#ff5e3a`, `rgba(255, 255, 255, .7)`
- *Icon sizes:* `8rem`
- *Line-heights:* `1.5`
- *Font-sizes:* `2rem`, `1.5rem`, `1.125rem`
- *Gaps:* `6vmin`
- *Margins:* `1.5rem`
- *Paddings:* `6vmin`, `1rem`
- *Border thicknesses:* `.2rem`
